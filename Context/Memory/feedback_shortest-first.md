name: shortest-first
description: When explaining something unfamiliar, give the shortest version first — expand only if asked
type: feedback
---

Lead with the minimal explanation that answers the question. Don't preemptively add context, caveats, or background.

**Why:** User had to sift through walls of text to find the 2–3 sentences that actually answered their question. Extra context made things harder to follow, not easier.

**How to apply:** Any time I'm explaining a concept the user said they don't understand — give the short version, stop, let them ask for more if needed.
