# Memory

_The AI's durable memory of how you like to work._

## What this folder is for

Preferences that should survive across sessions live here: your tone, formatting habits, and any standing rules. The AI reads them at the start of work and applies them quietly, so you don't have to repeat yourself.

## What lives here

Short notes capturing working-style preferences. When you tell the AI "always do it this way," this is where it remembers that.
