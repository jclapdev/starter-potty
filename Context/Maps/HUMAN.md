# Maps

_Index files that tell the AI where everything is and what it can do._

## What this folder is for

When the AI needs to know which skills exist, which agents are available, or where something lives in the vault, it reads a map here first. Maps keep the AI oriented as the vault grows.

## What lives here

Index notes such as a skill map, an agent map, and a vault map. The AI updates them automatically when it adds or changes things. Think of this folder as the table of contents for the whole system.
