## The handful of features you'll actually use

**The editor.** The middle of the screen. Type normally; Markdown adds light formatting: `#` for a heading, `-` for a bullet, `**bold**` for bold.

**The file explorer.** The left sidebar lists every file and folder. Click to open.

**Search.** The search icon in the left sidebar finds any word across every note at once.

**The command palette.** Run any Obsidian action by typing its name. When you can't remember a shortcut, this is how you find it.

**The graph.** A map of your notes and the links between them. Fun, optional, better once the vault grows.

**Plugins.** A big library of optional add-ons. You don't need any of them; this vault works out of the box.

Official help: https://help.obsidian.md

## Shortcuts already set up for you

`Mod` means **Cmd** on a Mac and **Ctrl** on Windows or Linux.

### Writing and formatting

| Shortcut | What it does |
|---|---|
| `Mod + N` | New note in the current tab |
| `Mod + Alt + L` | Insert a link to another note (`[[ ]]`) |
| `Mod + Alt + B` | Turn the line into a bullet point |
| `Mod + -` | Strikethrough |
| `Mod + \`` | Inline code (small `code` span) |
| `Alt + \`` | Code block (fenced, multi-line) |
| `Mod + Q` | Insert a footnote |

### Headings

| Shortcut | What it does |
|---|---|
| `Mod + 1` … `Mod + 6` | Set the line to Heading 1 through Heading 6 |
| `Mod + Alt + H` | Cycle the heading level up |

### Folding and line numbers

| Shortcut | What it does |
|---|---|
| `Mod + Alt + A` | Fold all sections (collapse) |
| `Mod + Alt + S` | Unfold all sections (expand) |
| `Mod + Alt + N` | Toggle line numbers |

### Layout

| Shortcut | What it does |
|---|---|
| `Mod + ←` | Show/hide the left sidebar |
| `Mod + →` | Show/hide the right sidebar |
| `Mod + D` | Split the editor so two notes sit side by side |
| `Mod + =` | Toggle stacked tabs |

### Changing or adding shortcuts

Settings, then Hotkeys. Search any action and set your own key. Full guide: https://obsidian.md/help/hotkeys
