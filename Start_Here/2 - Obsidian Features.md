# 2 - Obsidian Features

_[[Home]] · Back: [[1 - What This Is]] · Next: [[3 - Keyboard Shortcuts]]_

---

You only need a few features to get going. Here they are, in the order you'll meet them.

## The editor

The middle of the screen is where you write. Type normally. Markdown adds light formatting: `#` for a heading, `-` for a bullet, `**bold**` for bold. You don't have to memorize it; the [[3 - Keyboard Shortcuts]] page covers the ones already set up for you.

## The file explorer

The left sidebar lists every file and folder in your vault. Click a file to open it. This is how you move around by hand. The [[4 - Folders and Backlinks]] page explains how this vault is organized.

## Search

Press the search icon in the left sidebar to find any word across every note at once. For a faster jump to a specific note by name, use the quick switcher (covered in shortcuts).

## Links between notes

Type `[[` and Obsidian offers to link to another note. These links are the heart of Obsidian: they connect related notes so you can follow a train of thought. The [[4 - Folders and Backlinks]] page goes deeper, including **backlinks**, which show you every note that points at the one you're reading.

Official reference: https://help.obsidian.md/links

## The graph

Obsidian can draw a map of your notes and the links between them. It's optional and more fun than essential early on, but it's there when your vault grows.

## The command palette

Press the command palette shortcut and you can run any Obsidian action by typing its name. When you can't remember a shortcut, this is how you find it.

## Plugins (for later)

Obsidian has a large library of optional add-ons. You don't need any of them to start; this vault works out of the box. Explore them once you're comfortable.

Official reference: https://help.obsidian.md
